## Overviev
ideaMarket is a digital marketplace for buying and selling startup concepts, MVP hypotheses, and creative solutions. Built with Django REST Framework (Python 3.11) on the backend and React 18 with TypeScript on the frontend, the platform uses PostgreSQL for data storage, Redis for caching, and Stripe Connect for secure split payments between sellers and the platform. Key features include JWT authentication, idea catalog with filters, rating/review system, admin moderation panel, and Docker-based deployment with CI/CD via GitHub Actions. The project is MIT-licensed and open for contributions.
